IDE utilizado:

Visual Studio


Versión del lenguaje de programación utilizado:

C#,SQL,HTML,CSS


DBMS utilizado y su versión:

SQL Server 20



Lista de pasos para correr su aplicación:


a. Abre Visual Studio.

b. Abre tu proyecto de ASP.NET MVC en Visual Studio.

c. Asegúrate de que tu proyecto esté configurado para usar SQL Server como su base de datos. Esto se puede hacer mediante la configuración de la cadena de conexión en el archivo web.config o appsettings.json dependiendo de cómo hayas configurado tu aplicación.

d. Asegúrate de que tienes una instancia de SQL Server en ejecución y que tu base de datos esté disponible.

e. En el Explorador de soluciones de Visual Studio, haz clic derecho en tu proyecto y selecciona "Build" para compilar el proyecto.

f. Una vez compilado sin errores, puedes presionar F5 o seleccionar "Start Debugging" desde el menú de Visual Studio para ejecutar la aplicación.

g. Visual Studio abrirá un navegador web y cargarás la aplicación. Si todo está configurado correctamente, deberías poder interactuar con tu aplicación web.
